# YouTube video downloader

## Setup

```bash
$ virtualenv --python=python3.8 .env
$ source .env/bin/activate
(.env) $ python --version
Python 3.8.5

(.env) $ pip install -r requirements.txt
(.env) $ chmod u+x run.sh
(.env) $ ./run.sh
...

(.env) $ deactivate
```
